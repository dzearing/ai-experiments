# Project: ui-kit

## Core Value

A comprehensive design system providing themeable, accessible React components for AI-powered applications.

## Current Milestone: v2.0 Coworker Design System Parity

**Goal:** Implement layout, card, and chat components to achieve visual parity with Coworker design system (coworker-demo.lovable.app/design-system)

**Target features:**
- react-layout package (14 layout components)
- react-cards package (5 card components)
- react-chat enhancements (5 new components)
- Copilot theme (light/dark/high-contrast)
- Extended typography tokens
- Brand flair gradients
