Animation System

Consistent motion that enhances user experience without distraction.

Animation Principles

  • Purposeful: Every animation should have a clear purpose
  • Fast: Keep animations snappy (150-350ms for most interactions)
  • Smooth: Use appropriate easing curves for natural motion
  • Consistent: Similar actions should have similar animations
  • Accessible: Respect prefers-reduced-motion preferences

/* Animation Variables */ --duration-instant: 0ms; /* No animation */ --duration-fast: 150ms; /* Micro-interactions */ --duration-base: 250ms; /* Most transitions */ --duration-slow: 350ms; /* Complex animations */ --duration-slower: 500ms; /* Page transitions */ /* Easing Functions */ --ease-linear: linear; --ease-in: cubic-bezier(0.4, 0, 1, 1); --ease-out: cubic-bezier(0, 0, 0.2, 1); --ease-in-out: cubic-bezier(0.4, 0, 0.2, 1); --ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55); /* Usage Example */ .button { transition: all var(--duration-fast) var(--ease-out); } .modal { animation: slideUp var(--duration-base) var(--ease-out); } /* Respect User Preferences */ @media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } }

Implementation Guide

Best practices for using the design system in your projects.

/* 1. Import the design tokens */ @import 'design-system/tokens.css'; /* 2. Use semantic color tokens for theming */ .component { background-color: var(--bg-secondary); color: var(--text-primary); border: 1px solid var(--border-default); } /* 3. Use spacing tokens for consistent layout */ .container { padding: var(--space-6); margin-bottom: var(--space-8); } /* 4. Apply typography scale */ .heading { font-size: var(--font-size-xl); font-weight: var(--font-weight-semibold); line-height: var(--line-height-tight); } /* 5. Add smooth transitions */ .interactive-element { transition: background-color var(--duration-fast) var(--ease-out), transform var(--duration-fast) var(--ease-out); } /* 6. Use elevation for depth */ .card { box-shadow: var(--shadow-base); } .card:hover { box-shadow: var(--shadow-md); } /* 7. Maintain consistency */ .button { height: 36px; /* Use consistent heights */ padding: 0 var(--space-4); border-radius: var(--radius-md); font-size: var(--font-size-sm); font-weight: var(--font-weight-medium); }
