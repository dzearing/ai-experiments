# BreadcrumbCollapsed

**Priority**: Medium

**Description**: Space-efficient breadcrumb that collapses middle items into a dropdown.

## Features

- Auto-collapse logic
- Dropdown expansion
- Touch-friendly
- Smooth transitions
- Priority items
- Custom threshold
- Responsive breakpoints
- Always show first/last
- Tooltip previews

## Use Cases

- Mobile navigation
- Long paths
- Limited space
- Responsive designs
- Complex hierarchies
