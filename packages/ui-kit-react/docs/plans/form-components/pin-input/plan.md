# PinInput

**Priority**: Low

**Description**: Segmented input for PIN or verification codes.

**Features**:
- Auto-focus next
- Paste support
- Masked input
- Numeric only
- Custom length
- Error states
- Resend option
- Timer display
- Copy protection

**Use Cases**:
- OTP verification
- PIN entry
- Security codes
- Two-factor auth
- Verification
