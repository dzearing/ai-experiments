# FormField

**Priority**: High

**Description**: Wrapper component that provides consistent layout for form inputs.

**Features**:
- Label management
- Required indicator
- Help text
- Error messages
- Success feedback
- Tooltip integration
- Spacing control
- Responsive layout
- Accessibility labels
- Custom slots

**Use Cases**:
- Form layouts
- Input grouping
- Consistent spacing
- Validation display
- Accessibility
