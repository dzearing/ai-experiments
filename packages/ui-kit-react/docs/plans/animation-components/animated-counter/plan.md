# AnimatedCounter

**Priority**: Medium

**Description**: Smoothly animates number changes with rolling or counting effects.

## Features

- Count up/down
- Duration control
- Easing functions
- Format support
- Decimal places
- Thousands separator
- Prefix/suffix
- Start on visible
- Pause/resume

## Use Cases

- Statistics display
- Score changes
- Progress numbers
- Dashboard metrics
- Loading percentages
