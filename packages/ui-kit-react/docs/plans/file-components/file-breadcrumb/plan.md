# FileBreadcrumb

**Priority**: High

**Description**: Shows the current file path with clickable segments for navigation.

## Features

- Path segment clicking
- Dropdown for long paths
- Copy path button
- Home/root shortcut
- Path editing mode
- Custom separators
- Responsive truncation
- History integration

## Use Cases

- File browser navigation
- Path display
- Location context
- Quick navigation
- URL breadcrumbs
