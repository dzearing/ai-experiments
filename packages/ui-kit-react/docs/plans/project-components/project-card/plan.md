# ProjectCard

**Priority**: High

**Description**: Summary card for projects with key metrics and actions.

**Features**:
- Project name/icon
- Description preview
- Member avatars
- Progress metrics
- Quick stats
- Recent activity
- Action menu
- Favorite toggle
- Update indicator
- Custom fields

**Use Cases**:
- Project dashboards
- Project selection
- Portfolio views
- Team overview
- Quick access
