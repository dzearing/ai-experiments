# Flexbox

**Priority**: High

**Description**: Flexible box layout component with intuitive props.

**Features**:
- Direction control
- Justify options
- Align options
- Wrap behavior
- Gap support
- Flex children
- Order control
- Responsive props
- Inline flex
- Debug outline

**Use Cases**:
- Component layouts
- Navigation bars
- Card layouts
- Form layouts
- Centering
