# ResizablePanel

**Priority**: Low

**Description**: Panels that users can resize by dragging borders.

**Features**:
- Drag handles
- Min/max sizes
- Collapse option
- Double-click reset
- Save preferences
- Keyboard resize
- Touch support
- Nested panels
- Orientation toggle
- Smooth animations

**Use Cases**:
- Split views
- IDE layouts
- File explorers
- Dashboard panels
- Adjustable layouts
