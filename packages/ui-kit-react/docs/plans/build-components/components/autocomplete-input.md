# AutocompleteInput

**Priority**: High

**Description**: Generic autocomplete component for any data source.

**Features**:
- Custom data sources
- Async loading
- Multiple selection
- Create new option
- Fuzzy matching
- Custom rendering
- Grouping support
- Accessibility
- Mobile support
- Cache management

**Use Cases**:
- Form fields
- Tag inputs
- User selection
- Location inputs
- Product search
