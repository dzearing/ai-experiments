# Drawer

**Priority**: High

**Description**: Slide-out panel from screen edges.

**Features**:
- Four directions
- Push/overlay modes
- Multiple sizes
- Swipe gestures
- Backdrop option
- Header/footer
- Scrollable content
- Nested drawers
- Responsive behavior
- State persistence

**Use Cases**:
- Navigation menus
- Filter panels
- Settings
- Form wizards
- Detail views
