# ScaleTransition

**Priority**: Low

**Description**: Scale-based animations for zoom effects and attention-grabbing entries.

## Features

- Scale from center
- Custom origin points
- Combined with fade
- Bounce effects
- Hardware accelerated
- Gesture triggers
- Responsive scaling
- State preservation
- Exit animations

## Use Cases

- Modal appearance
- Button feedback
- Card expansion
- Zoom effects
- Attention UI
