# SearchFilters

**Priority**: Medium

**Description**: Filter UI for refining search results with multiple criteria.

**Features**:
- Filter types
- Active filters
- Clear filters
- Filter count
- Collapsible groups
- Range sliders
- Date pickers
- Apply/reset
- Filter presets
- URL sync

**Use Cases**:
- E-commerce filters
- Search refinement
- Data filtering
- Report filters
- Advanced search
