# MarkdownEditor

**Priority**: Medium

**Description**: Markdown editor with live preview and syntax assistance.

**Features**:
- Split view
- Live preview
- Syntax highlighting
- Toolbar shortcuts
- Image upload
- Table helper
- Link checker
- Export options
- Vim mode
- Distraction-free mode

**Use Cases**:
- Documentation
- README files
- Technical writing
- Note taking
- Blogging
