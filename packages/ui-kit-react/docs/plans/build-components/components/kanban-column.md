# KanbanColumn

**Priority**: Medium

**Description**: Individual column component for kanban boards.

## Features

- Header customization
- Card count
- WIP limits
- Column actions
- Color coding
- Collapse/expand
- Sort cards
- Column stats
- Drop zones
- Empty states

## Use Cases

- Kanban boards
- Task states
- Pipeline stages
- Category columns
- Status groups

## Implementation Notes

This component should serve as a building block for kanban boards with customizable headers and card management.
