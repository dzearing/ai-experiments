# RangeSlider

**Priority**: Low

**Description**: Dual-handle slider for selecting min and max values.

**Features**:
- Two handles
- Min distance
- Overlap prevention
- Value labels
- Custom formatting
- Histogram display
- Stepped values
- Smooth dragging
- Accessibility

**Use Cases**:
- Price filters
- Date ranges
- Min/max selection
- Data filtering
- Range queries
