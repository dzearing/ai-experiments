# Sticky

**Priority**: Medium

**Description**: Makes elements stick to viewport edges while scrolling.

**Features**:
- Top/bottom stick
- Offset control
- Z-index management
- Boundary element
- State callbacks
- Responsive behavior
- Smooth transitions
- Debug indicators
- Performance optimized
- Polyfill support

**Use Cases**:
- Headers
- Sidebars
- Table headers
- Navigation
- Toolbars
