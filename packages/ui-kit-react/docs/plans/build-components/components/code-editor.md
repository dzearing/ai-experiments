# CodeEditor

**Priority**: Medium

**Description**: Code input with syntax highlighting and IDE features.

**Features**:
- Syntax highlighting
- Line numbers
- Auto-indent
- Bracket matching
- Multiple languages
- Theme support
- Find/replace
- Code folding
- Autocomplete
- Diff view

**Use Cases**:
- Code input
- Configuration editing
- Snippets
- Teaching
- Debugging
