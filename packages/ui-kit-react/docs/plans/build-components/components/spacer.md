# Spacer

**Priority**: Medium

**Description**: Flexible spacing component for pushing content.

**Features**:
- Fixed sizes
- Flexible growth
- Responsive sizes
- Axis control
- Min/max sizes
- Conditional display
- Debug mode
- Custom units
- Percentage based
- Viewport units

**Use Cases**:
- Layout spacing
- Pushing content
- Flexible gaps
- Alignment
- Responsive spacing
