# FloatingToolbar

**Priority**: Low

**Description**: Contextual toolbar that appears near selections.

**Features**:
- Text selection trigger
- Position near selection
- Action buttons
- Keyboard shortcuts
- Touch support
- Custom actions
- Animation
- Hide on scroll
- Theme support
- Responsive size

**Use Cases**:
- Text formatting
- Code actions
- Selection tools
- Quick edits
- Context actions
