# TimePicker

**Priority**: Medium

**Description**: Time selection input with various formats.

**Features**:
- 12/24 hour format
- Minute intervals
- Scroll selection
- Keyboard input
- AM/PM toggle
- Seconds option
- Time zones
- Min/max times
- Clear button

**Use Cases**:
- Time selection
- Scheduling
- Reminders
- Event timing
- Appointments
