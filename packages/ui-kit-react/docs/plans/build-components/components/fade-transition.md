# FadeTransition

**Priority**: Medium

**Description**: Smooth opacity transitions for content appearance and disappearance.

## Features

- Fade in/out
- Cross-fade between
- Duration control
- Delay support
- Easing functions
- Mount/unmount handling
- Interrupt safe
- CSS variable based
- Performance mode

## Use Cases

- Content switching
- Loading states
- Image transitions
- Modal overlays
- Hover effects
