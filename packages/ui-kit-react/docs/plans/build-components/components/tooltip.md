# Tooltip

**Priority**: High

**Description**: Small overlay showing helpful information on hover.

**Features**:
- Auto positioning
- Delay control
- Arrow options
- Max width
- Multi-line support
- Touch behavior
- Keyboard trigger
- Rich content
- Theme variants
- Animation options

**Use Cases**:
- Help text
- Abbreviations
- Icon labels
- Truncated text
- Additional info
