# FileExplorer

**Priority**: Medium

**Description**: A full-featured file browser with multiple view options and rich interactions.

## Features

- Grid/list view toggle
- Thumbnail previews
- Sort options
- Column customization
- Quick preview
- Batch operations
- Advanced search
- File properties panel
- Recent locations

## Use Cases

- File management interfaces
- Media browsers
- Document libraries
- Asset managers
- Cloud storage UIs
