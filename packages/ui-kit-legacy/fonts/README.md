# Fonts Directory

This directory should contain the following font files:

## Segoe UI Web Fonts

- segoeui-light.woff2
- segoeui-light.woff
- segoeui-regular.woff2
- segoeui-regular.woff
- segoeui-semibold.woff2
- segoeui-semibold.woff
- segoeui-bold.woff2
- segoeui-bold.woff

## Cascadia Code (Monospace)

- cascadia-code.woff2
- cascadia-code.woff

These fonts are referenced in the CSS but need to be added to this directory.

To obtain Segoe UI Web fonts, you can:

1. Download from Microsoft's CDN
2. Extract from Windows system fonts
3. Use a web font service

For Cascadia Code:

1. Download from: https://github.com/microsoft/cascadia-code
2. Or use Google Fonts version
