# AgentStatus

**Priority**: Medium

**Description**: Real-time status indicator for AI agent activity.

**Features**:
- Active/idle states
- Processing indicator
- Queue length
- Response time
- Error states
- Uptime display
- Resource usage
- Log access
- Restart option
- Health check

**Use Cases**:
- Monitoring dashboards
- Agent cards
- Status bars
- Admin panels
- Debugging
