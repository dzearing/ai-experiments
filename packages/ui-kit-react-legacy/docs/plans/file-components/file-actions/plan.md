# FileActions

**Priority**: Medium

**Description**: Contextual menu component for file operations.

## Features

- Common operations (copy, cut, paste)
- Rename inline
- Delete with confirmation
- Share options
- Open with menu
- Properties display
- Custom actions
- Keyboard shortcuts
- Permission-aware

## Use Cases

- File context menus
- Toolbar actions
- Right-click menus
- Touch interactions
- Batch operations
