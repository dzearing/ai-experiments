# CommandPalette

**Priority**: Medium

**Description**: Cmd+K style universal command interface for power users.

**Features**:
- Global hotkey
- Fuzzy search
- Command grouping
- Recent commands
- Nested commands
- Custom actions
- Preview pane
- Keyboard only
- Context aware
- Plugin support

**Use Cases**:
- Power user features
- Quick navigation
- Action shortcuts
- Developer tools
- Productivity
