# QuickView

**Priority**: Low

**Description**: Rapid preview overlay for content items.

**Features**:
- Fast loading
- Minimal UI
- Keyboard navigation
- Swipe between
- Close on escape
- Loading states
- Error handling
- Cache support
- Preload adjacent
- Custom renderers

**Use Cases**:
- File previews
- Product quickview
- Article previews
- Image previews
- Quick details
