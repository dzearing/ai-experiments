# Toast

**Priority**: High (Enhancement of existing)

**Description**: Temporary notification messages that appear and auto-dismiss.

**Features**:
- Multiple positions
- Stack management
- Action buttons
- Progress bars
- Pause on hover
- Swipe to dismiss
- Custom duration
- Animation types
- Icon support
- Theme variants

**Use Cases**:
- Success messages
- Error notifications
- Info updates
- Action confirmations
- System messages
