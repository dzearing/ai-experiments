# SplitView

**Priority**: Low

**Description**: Two-pane layout with adjustable divider.

**Features**:
- Horizontal/vertical
- Initial sizes
- Min/max panes
- Collapse panes
- Splitter styling
- Double-click behavior
- Responsive behavior
- State persistence
- Keyboard control
- Smooth dragging

**Use Cases**:
- Code editors
- File browsers
- Email clients
- Preview layouts
- Comparison views
