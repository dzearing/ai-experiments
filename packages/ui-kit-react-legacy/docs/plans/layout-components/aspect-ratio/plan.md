# AspectRatio

**Priority**: Medium

**Description**: Maintains consistent aspect ratios for content.

**Features**:
- Common ratios
- Custom ratios
- Max width/height
- Object fit control
- Responsive ratios
- Fallback content
- Loading states
- Overflow handling
- Debug overlay
- Print support

**Use Cases**:
- Image containers
- Video players
- Card layouts
- Thumbnails
- Responsive media
