# TextArea

**Priority**: High

**Description**: Multi-line text input with auto-resize and character counting.

**Features**:
- Auto-resize option
- Character counter
- Max length enforcement
- Line counter
- Placeholder
- Resize handle
- Word wrap control
- Syntax highlighting option
- Emoji support
- Paste formatting

**Use Cases**:
- Long text input
- Comments
- Descriptions
- Messages
- Code input
