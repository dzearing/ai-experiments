# AvatarGroup Component

## Priority
Medium

## Description
Displays multiple avatars in an overlapping stack.

## Features
- Max display count
- Overflow indicator
- Hover expansion
- Custom spacing
- Size consistency
- Tooltip names
- Click actions
- Animation
- Responsive sizing
- RTL support

## Use Cases
- Team members
- Participants
- Collaborators
- Group chats
- Shared items
