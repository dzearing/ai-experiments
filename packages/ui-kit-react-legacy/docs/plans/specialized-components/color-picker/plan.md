# ColorPicker

**Priority**: Low

**Description**: Color selection tool with multiple input methods.

**Features**:
- Color wheel
- RGB sliders
- HEX input
- HSL support
- Eyedropper
- Palette presets
- Recent colors
- Opacity control
- Contrast checker

**Use Cases**:
- Theme customization
- Design tools
- Preferences
- Branding
- Visualization
