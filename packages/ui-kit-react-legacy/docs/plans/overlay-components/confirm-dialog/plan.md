# ConfirmDialog

**Priority**: High

**Description**: Specialized dialog for confirmation actions.

**Features**:
- Title/message
- Confirm/cancel buttons
- Danger variant
- Custom button text
- Icon support
- Don't ask again
- Loading state
- Keyboard shortcuts
- Auto-focus
- Promise-based API

**Use Cases**:
- Delete confirmations
- Dangerous actions
- Save confirmations
- Exit confirmations
- Verification
