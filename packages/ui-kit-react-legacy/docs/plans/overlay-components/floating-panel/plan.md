# FloatingPanel

**Priority**: Low

**Description**: Draggable floating window within the application.

**Features**:
- Drag header
- Resize handles
- Minimize/maximize
- Close button
- Stack order
- Bounds constraint
- Snap to edges
- State save
- Multi-panel
- Focus management

**Use Cases**:
- Tool windows
- Inspectors
- Chat windows
- Video players
- Palettes
