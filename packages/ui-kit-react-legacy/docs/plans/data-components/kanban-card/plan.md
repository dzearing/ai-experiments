# KanbanCard

**Priority**: Medium

**Description**: Draggable card component for kanban boards.

## Features

- Rich content
- Quick edit
- Labels/tags
- Assignee display
- Due dates
- Progress bars
- Comments count
- Attachments
- Priority indicators
- Card actions

## Use Cases

- Task cards
- Story cards
- Issue tracking
- Content cards
- Work items

## Implementation Notes

This component should extend the base Card component to add kanban-specific functionality with drag-and-drop support.
