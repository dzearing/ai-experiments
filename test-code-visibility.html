OLD (Problematic) Styling

In light mode, the text color (gray-800) on background (gray-100) creates poor contrast:

function showCode() {
    // This text might be hard to read in light mode
    // due to gray-800 text on gray-100 background
    console.log("Hello, World!");
    return true;
}

NEW (Fixed) Styling

Better contrast with gray-900 text on gray-50 background:

function showCode() {
    // This text should be much more readable
    // with gray-900 text on gray-50 background
    console.log("Hello, World!");
    return true;
}
